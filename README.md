# PORTFOLIO-SRISTI

🌐 Sristi Portfolio — 3D AI Powered Website

🚀 A modern, interactive developer portfolio built with cutting-edge technologies, showcasing projects, skills, and achievements with a premium UI/UX experience.

🔗 Live Demo: https://portfolio-sristi.vercel.app/

**✨ Features**
🎯 Interactive 3D Hero Section (Three.js)
🌌 Particle Background & Smooth Animations
⚡ Lightning-fast performance (Vite + React)
🎨 Modern Glassmorphism UI
📱 Fully Responsive Design
📂 Projects, Skills, Achievements sections
📩 Contact Form Integration (Formspree)
🎯 Resume Download Option


**🛠️ Tech Stack**

Frontend

React.js
Vite
Tailwind CSS

3D & Animations

Three.js
Framer Motion
Lenis (smooth scrolling)

Backend (optional AI integration)

Node.js
Express.js
OpenRouter API

**🚀 Getting Started**

Clone the repository:

git clone https://github.com/SRISTI9/PORTFOLIO-SRISTI.git
cd PORTFOLIO-SRISTI
npm install
npm run dev

**🌍 Deployment**

Deployed using Vercel

npm run build
